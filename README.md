# COST Stock Return Forecasting

In this project, I analyzed Costco Wholesale Corporation (COST) stock data by building a simple linear regression model to forecast next-day stock returns using historical return and volume-based features, comparing performance against a baseline, and converting the predicted return into an estimated next-day stock price.

Software Used: Python, Jupyter Notebook, pandas, numpy, matplotlib, yfinance, scikit-learn

## Project Features

1. Collected historical COST stock data using yfinance
2. Analyzed price, return, and volume trends
3. Created return-based and volume-based predictors
4. Built a linear regression model to forecast next-day returns
5. Compared model performance against a zero-return baseline

## Key Takeaway

The model slightly outperformed the baseline, but the small improvement reflects the high noise-to-signal ratio in short-term stock returns.
